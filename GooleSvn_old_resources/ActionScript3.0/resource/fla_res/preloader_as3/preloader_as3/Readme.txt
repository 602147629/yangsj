Read the following before using the files within this archive.

This archive contains files that accompany the article, "Creating preloaders using ActionScript 3," in the Adobe Flash Developer Center:
http://www.adobe.com/devnet/flash/articles/preloader_as3.html

You'll find the following files inside the preloader_as3.zip file:

* index.fla - This file is to be used while following the tutorial in the article.
* index_complete.fla - This file is a completed version of the entire exercise.
* fescue.swf - This is the file that will be used to load in the the "index" file.

All files are compatible with Adobe Flash Professional CS5 and later.
